Cookie Law Bar show bottom or top bar to inform users that your website uses cookie according to EU law.

== Description ==

A simple and lightweight cookie law WordPress plugin for show information that your website uses cookie. Perfect for implementation of EU cookie law!

Live demo: [http://widgetsforsite.com/](http://widgetsforsite.com/)

Feel free to try our other widgets powered by <a href="https://widgetpack.com/">Widget Pack</a>.

= Features =

* It's free
* Adaptive design
* Zero load time
* Smooth slide bottom or top bar
* Changeable: position, message, color, button text

== Installation ==
Unpack archive to this archive to the 'wp-content/plugins/' directory inside of WordPress.
Activate the plugin.
That's all!

  * Maintain the directory structure of the archive (all extracted files should exist in 'wp-content/plugins/cookie-law-bar/')
